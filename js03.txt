二.三目运算符:
	表达式1?表达式2:表达式3;
	解释:
		当表达式1中的boolean类型为true时会执行表达式2;
		当表达式1中的boolean类型为false时会执行表达式3;
	案例:
		var a=5;
		var b=3;
		var c;
		c=a>b?a:b;
		console.log(c);会打印出来a和b中的比较大的值输出;
三.switch/case
	说明:switch/case必须为等值匹配,不能是一个范围的匹配;
	语法:
		switch(匹配表达式){
			case:匹配值1
				语句1;一般为输出语句;
				break;
			caes:匹配值2;
				语句2;一般为输出语句;
				break;
			case:匹配值3;
				语句3;一般为输出语句;
				break;
			default:
				语句3:
					
		}
	备注:
	     1.switch语句执行的结束方式有两种:
					1.从匹配值开始直到运行到switch语句的结尾结束;
					2.遇到break关键字的时候跳出switch语句执行结束;
		
	     2.switch/case和if/else的使用情况说明:
					1.当数据是一个范围判断的时候使用if/else语句进行;
					2.当数据是一个等值匹配的时候采用switch/case语句进行;
	     3.switch语句的书写规范:
					1.所有的括号必须成对出现;
					2.一句话独占一行,不能出现一行写了多条语句;
					3.代码缩进一定要有使用快捷键 ctr+shift+F;
	     4.if和switch的注意点:
					1.if后面的()不能省略,()后面必须加分号;
					2.if的{}必须有;
					3.if的嵌套不能出现3层;
					4.switch语句的default分支必须要有,避免产生歧义;
					5.把执行概率大的放在前面.
			
四.循环:
	1.while循环:
		语法: while(条件表达式){
			循环体,执行语句
		}
	执行顺序:先判断条件,执行语句,....当条件不满足时结束.
	案例:   var sum=0;
		var i=1;
		while(i<101){
			sum=sum+i;
			i++;
		}
		console.log(sum);
      	要点:
		1.循环变量要进行初始化.
		2.循环体中应该具有使循环趋近于结束的条件.
		3.每次循环前判断表达式,表达式成立之后,在执行循环体.
		4.循环体中可以有若干代码.
	2.do.....while()循环
		语法:do{
		     }while();
		执行顺序:先执行一次循环语句,在执行条件,条件成立继续执行,如果不成立,结束.
	while循环和do..while循环的区别
		1.当首次条件成立没有任何区别;
		2.首次条件不成立,while不会执行,do..while会继续执行一次.
	3.for循环
		语法:
			for(表达式1;表达式2;表达式3){
				执行的语句;
			}
	如何选择使用的循环:
		1.知道循环次数 使用for循环;
		2.不知道循环的次数,使用while循环;
		3.当至少需要执行一次循环时必须使用 do..while循环.

	6.break:
		1.switch/case中使用;
