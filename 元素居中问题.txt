通过弹性盒子布局让未知大小的元素在屏幕窗口居中
html,body{
	height:100%;	
}
body{
display:flex;
justify-content:center;
align-items:center;
}
通过定位实现未知大小的元素在屏幕窗口居中
元素{
position:fixed;
top:0;
left:0;
right:0;
bottom:0;
margin:auto;
}
通过css3变形的方式实现未知大小得元素在屏幕窗口居中
元素{
position:fixed;
top:50%;
left:50%;
transform:translate(-50%,-50%);
}

}
